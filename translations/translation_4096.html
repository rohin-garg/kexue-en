Scientific Spaces "WeChat Group | Chatbot" Online Testing

By 苏剑林 | November 24, 2016

I spent some time completing a WeChat chatbot and established a WeChat group.

The currently implemented functions are as follows:

1. Search for the WeChat ID spaces_ac_cn; after adding it as a friend, it will automatically send you a group invitation, and you can join the group chat once you accept;
2. Automatically send a welcome message after joining the group;
3. Record group chat logs and share them with everyone periodically, so you no longer have to worry about losing valuable group information;
4. If a group becomes full, a new group will be opened, and information from one group will be automatically synchronized to the other, so that no group is left isolated;
5. If you send a message to the WeChat ID spaces_ac_cn, it will automatically search for answers on Zhihu and return them; this is also a simple Zhihu search bot.

There are also some functions used by administrators, which I won't list in detail.

Everyone is welcome to join! Please provide feedback if you encounter any issues. There may be bugs in the code, so I hope everyone can help test it extensively.
